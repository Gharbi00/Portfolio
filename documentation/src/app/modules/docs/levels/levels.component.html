Setting Up Loyalty Levels

Introduction

Once you join Elevok, you can set up and configure loyalty levels through our application marketplace. Follow the steps below to customize your loyalty program under the "Customer Engagement" section, specifically in the "Levels" tab.

Configuration Options

  • Number of Levels: Define the total number of loyalty levels you want to create for your customers.
  • Color of Each Level. Assign a unique color to each level for easy differentiation and visual appeal.
  • Name of Each Level: Specify a distinctive name for each loyalty level to reflect its status and rewards.
  • Description of Each Level: Provide a rich text description for each level. You can enhance the description with Bold text, Colored text, Images and Other rich text formatting options
  • Picture of Each Level: Upload an image representing each loyalty level to make it more engaging and visually appealing.

Inactivity Penalty (Gamification Feature)

Add excitement to your loyalty program by setting up penalties for inactivity. For each level, you can:

  • Specify the number of reputation points a user will lose if they are inactive for a defined period.
  • Configure the exact number of days a user must be inactive before points are deducted.

Inactivity Configuration

In the "Inactivity" tab, you can fine-tune which activities are considered when calculating inactivity loss. This allows you to:

  • Define specific user activities that count towards activity, ensuring the inactivity penalty is fair and accurately reflects user engagement.

By following these steps and utilizing the customization options available, you can create a dynamic and engaging loyalty program tailored to your customer engagement strategy. For more detailed instructions and examples, refer to our detailed guides and FAQs in the Elevok Help Center.
