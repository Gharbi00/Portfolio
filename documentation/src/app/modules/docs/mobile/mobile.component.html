Introduction

Display your user's Elevok customer profile, inclusive of all details, reward campaigns, and leaderboard information within your application.

Elevok User Profile Sync

The initial step involves syncing customer profiles upon login or registration, allowing Elevok to track their progress and administer rewards effectively.

To sync customer profiles, utilize the Player Login API method, which facilitates the creation or updating of customer profiles with each call.

It's advisable to invoke the Player Login API during every login or account update process.

Depending on your application's architecture, profile sync API calls can be executed either from your backend system or directly within the app.

Displaying Elevok Customer Profile

The subsequent step is to showcase the Elevok customer's loyalty profile within your application, enabling users to monitor their progress, points, and rewards.

You have two options for integrating Elevok's loyalty profile into your mobile app:

  • Utilizing the pre-built widget.
  • Utilizing Elevok's extensive APIs to customize the UI elements as desired.

Elevok provides a pre-built widget that seamlessly integrates into your mobile app, particularly if your framework supports Webviews.

Let's illustrate the sequence to display the Elevok profile widget using a Webview:

  • Add a menu item to your app menu or customer's profile page.
  • Upon click, open a Webview pointing to a self-hosted webpage with embedded Elevok widget scripts.
  • Pass customer data to the Webview.
https://example.com/elevok-widget/index.html

For the mobile app scenario, ensure the 'embed:true' parameter exists in the loading script to accommodate mobile-specific requirements.

embed:true

Pass customer data to your Webview by either directly passing parameters if supported by the framework or through query strings while loading the Webview.

<!-- index.html -->
  
  <!DOCTYPE html>
  <html>
    <head>
      // Your head content ........
    
      <script src="https://widget.elevok.com/sdk.js" defer="defer"></script>
    </head>
    <body>
      // Your body content ........
    
      <script> window.widgetInit = { appId: 'Your_Client_Key', locale: 'en', theme: 'light' };</script>
    </body>
  </html>

By following these steps, you can seamlessly pass customer data to the Elevok widget script and provide users with a personalized experience.

If you opt for using the pre-built widget in a Webview, ensure smooth Webview interactions within your app, such as navigating to and from the Webview. Consider displaying the Webview as a modal with a back button for enhanced user experience.
