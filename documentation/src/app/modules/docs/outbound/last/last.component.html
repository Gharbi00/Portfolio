Get the last widget outbound message

Introduction

This endpoint retrieves the latest widget outbound data for a specific target extracted from the authorization token.

Endpoint

URL: /outbound/last
Method: POST

Request Headers

Here you need to insert the bearer token obtained from the Login and Get Access Token API.


  {
    "Authorization": "Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
  }

Request Parameters

reference*: Session reference found in local storage or cookies under key elevkusr.

Note: In case you don't find this reference you can skip it because it's optional. Before showing the outbound plese verify if you have the identifier in your cookies with the name you choose when saving it.

Error Handling

404 Not found: In case there is no inbound information to show to the user.

Sample Request

Upon successful authentication, the API returns an access token.


POST /outbound/last
Authorization: Bearer YOUR_ACCESS_TOKEN
{
  "reference": "asdfa87asdf"
}

Sample Response

Upon successful request, the API returns a confirmation.
