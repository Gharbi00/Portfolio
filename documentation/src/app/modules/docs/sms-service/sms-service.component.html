SMS Integration Configuration

Introduction

To send SMS notifications and marketing messages to your users, you need to configure your SMS integration settings. This section will guide you through the steps to set up SMS integration within Elevok. You can find these settings in the Elevok Marketplace under the SMS Integration app.

Gateway Selection

Choose the SMS gateway you want to use for sending messages.

Setting Up Gateway

  • Key: Provide the API key or authentication token required by your selected SMS gateway.
  • Sender: Specify the sender name or number that will appear on the recipient's device.

Example Configuration

  • Gateway: Twilio
  • Key: Your Twilio API key.
  • Sender: Your Twilio phone number

SMS Sending

  • After configuring your SMS integration settings, you can start sending SMS notifications and marketing messages to your users.
  • Test your SMS setup to ensure that messages are delivered as expected.

By properly configuring your SMS integration settings, you can effectively communicate with your users via text messages. For more detailed instructions and troubleshooting tips, please refer to the Elevok Help Center.
