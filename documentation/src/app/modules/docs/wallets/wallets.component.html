Wallet Settings Page

Introduction

The Wallet Settings page allows customers to configure their coins and points. Here, you can set up two types of points to reward consumer actions: quantitative (monetary value) and qualitative (status and levels value). You can access this page in the Elevok Marketplace under the Customer Engagement app, specifically in the Wallets tab.

Configuring Points

  • Quantitative Points (Monetary Value): Select the type of quantitative points you want to use for rewarding actions. These points have a direct monetary value.
  • Qualitative Points (Status and Levels Value): Enable the qualitative points to reward actions based on status and levels and choose an avatar to represent the qualitative points.

Creating and Managing Coins

You have two options for managing coins in your wallet, you either create your own coin or use one of our aggregators

Creating Your Own Coin

Create a custom coin by following these steps:

  • Upload an avatar for the coin.
  • Choose a picture that represents the coin.
  • Define the value of the coin in fiat currency.
  • Save the coin configuration. This custom coin will now be available for rewarding consumer actions.

Using Aggregator Coins

If you prefer not to create your own coin, you can select from one of our aggregators in the dropdown menu. This option allows you to:

  • Use an existing coin from the aggregator for rewarding quantitative points.
  • Configure settings according to the selected aggregator's specifications.

By configuring your wallet settings, you can tailor the reward system to fit your business needs and enhance customer engagement effectively. For more detailed guidance on each option, please refer to the Elevok Help Center.
