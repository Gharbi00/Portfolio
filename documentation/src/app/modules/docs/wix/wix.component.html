Integrating with Wix

Introduction

Elevok seamlessly integrates with Wix, allowing you to enhance your customer engagement and loyalty programs directly within your Wix store. Follow these simple steps to integrate Elevok into your Wix platform

Installing the Elevok App

Navigate to the Wix store from your Wix admin dashboard.

Search for the Elevok app within the Wix store.

Click on the Elevok app listing and proceed with the installation process.

Configuration

After installation, access the Elevok app settings within your Wix admin panel.

Configure the Elevok app by providing the following parameters:

  • App ID: Unique identifier for your Elevok integration.
  • Public Key: Key used for authentication and authorization.
  • Secret Key: Secure key for accessing Elevok services.

Save the configuration settings to apply changes.

Widget Deployment

Upon saving the configuration, the Elevok widget will automatically appear on your Wix website. This widget serves as the interface for your customers to interact with Elevok's loyalty features.

Utilizing the Plugin

With the Elevok integration, you gain access to a plugin that facilitates the use or deduction of points during the checkout process. This plugin is designed to seamlessly integrate into your Wix checkout flow, enhancing the overall customer experience.

Customization Options

Elevok offers customization options to tailor the appearance of the loyalty plugin according to your branding and design preferences.

Access the configuration settings of the Elevok plugin within your Wix admin panel.

In the "Custom Style" field, input custom CSS to modify the visual aspects of the loyalty plugin. Below is an example of how you can format your custom CSS:

{
      margin-top: 20px;
      margin-bottom: 20px;
  }
  
  .wallet-label {
      display: inline-block;
      width: 330px; 
  }
  
  .coins-label {
      display: inline-block;
      width: 320px; 
  }
  
  #input-coins-step {
      margin-bottom: 10px; 
  }
  
  #apply_discount_btn_wrapper {
      overflow: hidden; /* Clear float */
  }
  
  #apply_discount_btn {
      float: right; /* Float the button to the right */
  }
  
  .apply-btn {
      background-color: #007bff; /* Blue */
      border: none;
      color: white;
      padding: 5px 20px;
      text-align: center;
      text-decoration: none;
      display: inline-block;
      font-size: 16px;
      margin: 4px 2px;
      cursor: pointer;
      border-radius: 4px;
  }

By following these steps, you can seamlessly integrate Elevok into your Wix store, empowering you to enhance customer engagement, loyalty, and retention directly within your eCommerce platform.
